Welcome to the Transformers API

Your place for all your Transformers data needs

Autobot Routes:

  • GET /api/autobots - Retrieve all Autobots
  • GET /user - Retrieve a single user and their associated Autobots Requires Authentication
  • POST /api/autobots - Add an Autobot to the database JSON {name: STRING, color: STRING} | Requires Authentication
  • PUT /api/autobot - Update an Autobot in the database JSON {autobot_id: ID, name: STRING, color: STRING} | Requires Authentication
  • DELETE /api/autobot - Remove an Autobot from the database JSON {autobot_id: ID} | Requires Authentication

User Routes:

  • POST /api/auth/register - Register a user account JSON {email: STRING, password: STRING(minimum 6 characters)}
  • POST /api/auth/login - Log in to your accountJSON {email: STRING, password: STRING}
